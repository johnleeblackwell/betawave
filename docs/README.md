# βWave documentation

Module guides — each one covers setup, the daily workflow, and the safety rails:

| Guide | Covers |
|---|---|
| [Syndicate](syndicate.md) | Auto-reposting to X, Facebook, Instagram, Telegram, Reddit, Medium — incl. full Meta app setup |
| [Discovery](discovery.md) | Prospecting CRM ranked by AI invisibility — verticals, CSV imports, outreach pipeline |
| [Site](sites.md) | Client websites (Netlify or WordPress), pSEO content engine, draft-safe publishing |
| [Citations](citations.md) | Tracking brand mentions in ChatGPT/Claude/Perplexity/Gemini answers + the gap-closer |
| [Respond](respond.md) | Human-gated engagement inbox — AI drafts, humans approve, nothing auto-sends |
| [Settings](settings.md) | BYO API keys (encrypted), local inference, owner/operator roles, white-labelling |

Installation and instance configuration live in [INSTALL.md](../INSTALL.md).

## The loop these modules form

```
Discovery finds who needs you → Citations proves the gap → Site publishes the content
that closes it → Syndicate amplifies it everywhere → Respond handles what comes back
→ Citations measures the improvement → repeat
```
